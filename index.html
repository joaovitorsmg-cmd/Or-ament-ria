📌 Como atualizar
🐍 Via script Python (recomendado — inclui nomes de filial/regional/auditor):
Coloque na mesma pasta os 3 arquivos:
  📄 Mapa_Despesa.xlsx  +  📄 Base_regionais.xlsx  +  🐍 gerar_json.py
  1. Baixe o gerar_json.py e o gerar_json.bat abaixo
  2. Coloque os 3 arquivos na mesma pasta que os Excel
  3. No Windows: dê 2 cliques em gerar_json.bat — aguarde o script finalizar
  4. Arraste o aqm_data.json gerado na área acima
✨ Alternativa — direto pelo Excel (sem Python):
  1. Arraste o Mapa_Despesa.xlsx e o Base_regionais.xlsx juntos na área acima
  2. O painel atualiza automaticamente com os nomes completos
